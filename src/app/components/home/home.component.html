Karma

is the default test runner for applications created with the Angular CLI.

testing frameworks

  • Jasmine

    is a behavior-driven development framework for testing JavaScript code that plays very well with Karma.

    Jasmine is also dependency free and doesn’t require a DOM.

    The most notable example would be spies. A spy allows us to “spy” on a function and track attributes about it such as whether or not it was called, how many times it was called, and with which arguments it was called. With a framework like Mocha, spies are not built-in and would require pairing it with a separate library like Sinon.js.
